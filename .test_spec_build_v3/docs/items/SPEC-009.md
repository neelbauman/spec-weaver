# [SPEC-009] 一覧テーブルのフィルタリング機能

**実装状況**: 📝 draft

**上位アイテム**: [REQ-008](REQ-008.md)

**テスト対象**: Yes　**個別カバレッジ**: 🟢 1/1 (100%)


### 内容

## 概要
`build` コマンドで生成される一覧ページのテーブルに、動的なフィルタリング機能を追加する。

## 詳細仕様
- MkDocs の `tablesort` などのプラグイン、あるいは DataTables 等のライブラリを利用し、
  ブラウザ上でテーブルのフィルタリングとソートを可能にする。
- 以下の項目をフィルタリング対象とする：
    - ID
    - タイトル (Header)
    - ステータス (Status)
    - レベル (Level)
    - 親
    - 子
    - 兄弟
- ユーザーが入力したキーワードによる部分一致検索をサポートする。
- 親、子、兄弟、ステータスについては、ドロップダウンなどによる選択形式のフィルタも検討する。
