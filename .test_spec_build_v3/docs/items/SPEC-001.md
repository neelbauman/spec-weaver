# [SPEC-001] コア・アーキテクチャ

**実装状況**: ✅ implemented

**上位アイテム**: [REQ-001](REQ-001.md) / **兄弟アイテム**: [SPEC-002](SPEC-002.md)


### 内容

## 概要
Spec-Weaverのコアアーキテクチャに関する技術的制約事項を定義する。

## 詳細仕様

### 言語・フレームワーク
- Python 3.10+ を使用すること
- CLIフレームワークとして `Typer` を使用すること
- ターミナルUIとして `Rich` を使用すること

### 責務の分離（設計原則）
- `spec-weaver` 自身はHTMLのレンダリングやMermaidの描画エンジンを持たないこと
- データの抽出・結合（Markdown生成）に徹し、描画の全責務を `MkDocs` に委譲すること
- 「データ構築 vs 描画」の分離を厳守すること
